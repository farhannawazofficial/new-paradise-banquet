## ğŸ“ Overview

**New Paradise Banquet** is a digital solution designed to establish a strong online presence for the banquet hall. This project serves as a central hub for potential clients to view the venue, check services, and make inquiries.

The goal of this repository is to provide a seamless user experience (UX) for visitors looking for event venues for weddings, parties, and corporate functions.

## âœ¨ Key Features

* **Responsive Design:** Fully optimized for mobile, tablet, and desktop devices.
* **Elegant Gallery:** High-quality image showcase of the hall, decor, and past events.
